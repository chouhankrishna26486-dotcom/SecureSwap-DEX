# ?? SGSU Smart Attendance & AI Chatbot System

This is a web-based application developed for **Scope Global Skills University (SGSU), Bhopal**. It combines a smart attendance verification system with an AI-powered enquiry chatbot.

## ? Key Features

### 1. ? Smart Attendance System
- **Camera Access:** Captures live video feed for student verification.
- **GPS Location Tracking:** Verifies if the student is physically present at the campus (NH-12, Misrod).
- **Real-time Verification:** Instant feedback using browser APIs.

### 2. ? AI Campus Chatbot
- **24x7 Support:** Answers queries regarding Admission, Fees, and Placements.
- **Course Expert:** Provides detailed syllabus and future scope for B.Tech (CSE, AI/ML, Mechanical), BCA, and Management courses.
- **Official Contacts:** Provides real SGSU contact numbers, email IDs, and Google Maps location.

### 3. ?? Interactive UI
- **Campus Slideshow:** Displays dynamic images of the SGSU campus.
- **Official Branding:** Uses the official SGSU logo and color theme.

## ?? Tech Stack Used
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **APIs:** Geolocation API, MediaDevices API (Webcam)
